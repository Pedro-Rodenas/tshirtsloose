Pedidos y Entregas

¿Cómo funciona?

¿Cómo hacer un pedido?

Para poder realizar un pedido, simplemente sigue estos pasos sencillos:

1. Selecciona los productos que deseas comprar de nuestro catálogo online.

2. Añádelos a tu carrito de compras y revisa tu pedido.

3. Procede al pago de forma segura con trato directo.

4. Recibirás un mensaje de confirmación.

¿Cómo se entregará?

El proceso de entrega se realiza de manera rápida y eficiente, tan pronto como se confirme el pago. Los pasos incluyen:

1. Una vez recibido el pago, tu pedido será preparado para el envío.

2. Utilizamos servicios de mensajería confiables para garantizar que recibas tus productos a tiempo.

3. Recibirás una notificación con los detalles del envío y el tiempo estimado de entrega.
